<head> <b> Deploying Custom YOLOV5 Custom Trained Models </b></head>


Clone the project

```bash
  https://github.com/ImtiazUlHassan/yolov5withCustomisedDetect.py.git
  
```

Go to the project directory

```bash

cd yolov5

pip install -r requirements.txt  # install

```
Run the following 

```bash

python3 Webcam.py

```
